Exploration

First of all, I want to explore various definitions and structures in math. This will flow pretty mathematically rather than with words.

Definition (Magma). Consider a set $S$ with (closed) binary operation $\times$. Then $S$ is called a magma.

Definition (Semigroup). A magma with an associative operation is a semigroup.

Definition (Monoid). Consider a set $S$ with binary operation $\times$, $S$ is a semigroup. If $S$ also satisfies

  • $\exists 1 \in S$ such that $\forall s \in S, 1 \times s = s \times 1 = s$
then we call $S$ a monoid under $\times$. $1 \in S$ is called the identity of $S$ with respect to $\times$.

Remark. We say "the" identity because it can be shown that the identity is unique: if $s_1,s_2$ satisfy $se_1 = e_1s = s$ for all $s$ and $se_2 = e_2s = s$ for all $s$, then $e_1 = e_1 e_2 = e_2$.
Remark. Sometimes we use multiplicative notation, which means that we write $ab$ instead of writing $a\times b$. Here, the operation is implicit.

Definition (Group). Consider $S$ with binary operation $\times$. If $(S, \times)$ is a monoid with identity $1 \in S$ and $\times$ also satisfies:

  • $\forall s \in S, \exists s^{-1} \in S$ such that $ss^{-1} = s^{-1}s = 1$
then we call $S$ a group under $\times$. For all $s$, $s^{-1}$ is called the inverse of $s$.

Remark. We say "the" inverse because it can be shown that the inverse is unique: if $s_1,s_2$ satisfy $ss_1 = s_1s = ss_2 = s_2s = 1$, then $s_1 = s_1 1 = s_1(ss_2) = (s_1s)s_2 = 1s_2 = s_2$.

Definition (Abelian/Commutative). Suppose a binary operation $+ : S \times S \rightarrow S$ satisfies $a + b = b + a$ for all $a, b \in S$. Then we say the operation is abelian or commutative.

Definition (Abelian Monoid). We say a monoid is abelian if its operation is commutative.

Definition (Abelian Group). We say a group is abelian if its operation is commutative.

Definition (Rig/Semiring). Consider a set $S$ with two binary operations $+$ and $\times$. If $S$ is an abelian monoid under $+$ with identity $0 \in S$ and $S$ is a monoid under $\times$ with identity $1 \in S$, and the following distributive properties hold:

  • $\forall a,b,c \in S$, $a \times (b + c) = a \times b + a \times c$
  • $\forall a,b,c \in S$, $(b + c) \times a = b \times a + c \times a$
and additionally,
  • $0 \times a = a \times 0 = 0$ for all $a \in S$
then $S$ is called a rig or semiring.

Remark. It is called a rig because "negatives" (or inverses with respect to $+$) don't necessarily exist.

Definition (Rng/Non-unital ring). Consider a set $S$ with two binary operations $+$ and $\times$. If $S$ is an abelian group under $+$ with identity $0 \in S$ and $S$ is a semi-group under $\times$, and the following distributive properties hold:

  • $\forall a,b,c \in S$, $a \times (b + c) = a \times b + a \times c$
  • $\forall a,b,c \in S$, $(b + c) \times a = b \times a + c \times a$
then $S$ is called a rng or non-unital ring.

Definition ((Unital) Ring). A ring or unital ring is an rng such that $S$ is a monoid under $\times$ with identity $1 \in S$. Equivalently, a ring is a rig such that $S$ is an abelian group under $+$.

Remark. The last condition that $a\times 0 = 0 \times a = 0$ is no longer needed after introducing additive inverses.

Definition (Commutative Ring). A commutative ring is a ring such that $S$ is an abelian monoid under $\times$.

Definition (Division Ring). A division ring is a ring in which $S/\{0\}$ is a group under $\times$.

Theorem (Wedderburn's Little Theorem). Any finite ring which is a group with respect to $\times$ is also an abelian group with respect to $\times$, or a finite field.
The proof to this theorem seems to be a bit more complicatd than I would like to get in this post, so I'll explore it in a future post.

Definition (Field). A field is a commutative ring where $0 \neq 1$ and $S$ is an abelian group under $\times$.

Definition (Galois Field). A Galois field is a field which is finite.

The Theorem

The question is... how many operations can we have at the same time on a set? This sounds weird, but the question is inspired by the fact that all of the algebraic structures I listed above only have at most two operaitons. Why not more? Even more interesting, why do we not have a structure which is a set $S$ such that $S$ with $+$ is a monoid and $S$ with $\times$ is a monoid? One explanation is that it may not be very applicable, but a post I saw on Stack Exchange claimed otherwise. Maybe I'll look into this in more detail in a future post.

For this post, however, I want to consider a finite set $S$ and consider a structure where $S$ is an abelian group with respect to $+$ with identity $e_+$, $S/\{e_+\}$ is an abelian group with respect to $\times$ with identity $e_\times$ (and a monoid when $e_+$ is included), and $S/\{e_\times\}$ is an abelian group with respect to $*$ with identity $e_*$ (and a monoid when $e_\times$ is included). Additionally, we will impose the following distributive laws: $$\forall a,b,c \in S, a \times (b + c) = a \times b + a \times c$$ $$\forall a,b,c \in S, a * (b \times c) = a * b \times a * c$$ and the condition that $e_+ \neq e_\times \neq e_*$.

This is essentially a generalization of a Galois field to three operations.

Theorem. Suppose $S$ has $p$ elements for some prime $p$.
Proof.

Since $S$ has $p$ elements, by a well-known theorem, $S$ is isomorphic to $\mathbb{Z}/p\mathbb{Z}$.
